<h1 align="center">WebLite</h1>

WebLite is a simple proxy made to make websites lightweight by stripping out unnecessary tags from the page.

## Technologies Used
- Python
- Flask
- Readability-lxml
- BeautifulSoup4

## Running the App

1. Clone the repo.

2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3. Run the app:
    ```bash
    python app.py
    ```


A live demo of the app is available at [https://web-lite-omega.vercel.app/](https://web-lite-omega.vercel.app/)
